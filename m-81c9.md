# Grammar: multi-line function calls inside blocks fail to parse

Multi-line function calls work at top level but fail inside if/when blocks. Example that fails:

```mlld
if true [
  let @result = @test(
    "one",
    "two"
  )
]
```

Error: Expected end of input but "(" found
