# Array.includes() returns false for values that == true

**Problem**

```mlld
var @polishArg = "high"
var @validPolish = ["false", "med", "high"]
var @isValid = @validPolish.includes(@polishArg)
show @isValid  // false

var @isHigh = @polishArg == "high"
show @isHigh  // true
```

The value equals "high" via `==` but `includes()` returns false.

**Workaround**

Use `when` with explicit cases instead of array includes.

**Discovered during**

j2bd --polish argument validation.
