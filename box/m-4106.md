# Phase 7: Agent config extraction to registry modules

Spec: `_specs/box.md` (Part 6: Agent config modules)

Extract agent-specific knowledge (Claude, Codex) from mlld core into importable registry modules. Refactor CLI commands to use module-based architecture instead of baked-in agent logic.

## Context

Phase 1 handles the mechanical rename of CLI commands/files (env.ts→box.ts, command names, imports). This phase is about the architectural refactoring: extracting agent setup into composable modules and updating CLI commands to pull from registry instead of containing agent-specific logic directly.

## CLI Refactoring (NOT rename — Phase 1 does the rename)
- `mlld box capture` → refactor to: discover agent type, pull module from registry, run auth setup, generate local `.mlld/box/{name}/index.mld`
- `mlld box spawn` → refactor to load generated module instead of hardcoded logic
- `mlld box shell` → refactor similarly
- Directory paths: `.mlld/env/` → `.mlld/box/`
- Help text updated in `cli/interaction/HelpSystem.ts` to reflect module-based workflow

## Agent Config Modules (registry)
Create registry modules that export agent setup as composable mlld values:

### @mlld/agents/claude
```mlld
var @settings = {
  "permissions": {
    "allow": ["Bash(*)", "Read(*)", "Write(*)"]
  }
}

var @claude-md = "You are working in a sandboxed mlld workspace..."

export { @setup }
var @setup = [
  { "settings.json": @settings, desc: "Claude Code permissions" },
  { "CLAUDE.md": @claude-md, desc: "Agent instructions" }
]
```

### @mlld/agents/codex (when ready)
Similar structure for Codex config.

### @mlld/agents/base
Shared agent config utilities.

## Usage Pattern
```mlld
import { @setup } from "@mlld/agents/claude"
import { @my-hooks } from "./my-hooks.mld"

var @my-setup = @setup + [
  { "hooks.json": @my-hooks, desc: "Custom safety hooks" }
]

box [
  files ".claude/" = @my-setup
  file "task.md" = @task
  exe @agent
]
```

## Auth Handling
Auth extraction becomes a module-level concern:
```mlld
import { @setup, @configure-auth } from "@mlld/agents/claude"
var @auth = exe @configure-auth
box { auth: @auth } [
  files ".claude/" = @setup
  exe @agent
]
```
