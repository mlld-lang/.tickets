# Create llm/lib/common.mld shared library

## Problem

@fileExists, @mkdirp, and @flatName are duplicated across 4-5 orchestrator scripts with identical implementations.

## Duplications

- @fileExists: review/lib/context.mld, doc-audit/lib/context.mld, j2bd/lib/context.mld, review-docs/lib/context.mld
- @mkdirp: review/lib/context.mld, j2bd/index.mld, doc-audit/index.mld, review-docs/index.mld, doc-writer/index.mld
- @flatName: review/lib/context.mld, review-docs/lib/context.mld

## Work

1. Create llm/lib/common.mld with @fileExists, @mkdirp, @flatName
2. Export all three
3. Update all orchestrators to import from @lib/common instead of local definitions
4. Remove local duplicates

See llm/CONVENTIONS.md for the @lib/ path alias convention.
